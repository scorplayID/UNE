Hi nama saya asli saya M.caesar.F yang mempunyai Scorplay dan Tim unie ini. Saya akan menceritakan bagaimana pembentukan tim union ini dan apa alasan saya untuk membentuk tim sendiri.. ... Pada akhir tahun 2024 saya memiliki ide Untuk membuat tim sendiri, sebelumnya saya menjadi anggota tim Kerang ajaib yang di buat oleh @bang_oxa, lalu Saya mengajak player Random yaitu @Daungaming untuk memenuhi syarat sebagai tim union lalu Saya mengajak @Craftsman @Alesha lalu saya membentuk grup WhatsApp pada 28 april 2025 dan saya mendaftarkan diri sebagai ketua tim tetapi pendaftaran sebagai ketua tim belum di buka jadi saya mengunggu pengumuman kapan di buka, lalu di akhir april ada masalah ke handphone saya karena masalah layar dan saya berharap data data aman karena semua data penting di sana lalu saya memakai handphone lain untuk tetap berkomunikasi dengan calon anggota tim, dan 1 bulan pun tiba di bulan Juli handphone pun kembali dengan data aman, lalu saya memulai pembentukan tim, mulai dari nama tim dan kesiapan mereka untuk menjadi anggota, nama pertama tim ini adalah Craftmingplay atau di singkat menjadi CMP lalu salah satu dari mereka keluar dan saya takut ada yang keluar lagi dan saya mempromosikan kembali tim CMP lalu di bulan September terdapat anggota baru yaitu @Alarc dan beberapa bulan kemudian sistem union berubah kita tidak perlu menunggu jadwal pendaftaran sebagai anggota tim, dan saya mengubah nama tim CMP Menjadi UNIE Lalu beberapa bulan kemudan saya mencoba menambah anggota baru dan ada 2 anggota yang berminat dan sampai sekarang mereka masih menjadi anggota tim unie ♥️,

Lalu apa alasan saya membetuk anggota tim?
Alasan saya membetuk anggota tim adalah untuk saling membantu di uni, agar mereka tidak sendirian untuk melakuan apapun di uni.
